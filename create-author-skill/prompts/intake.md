# Intake Prompt

Collect only the following fields:

1. `slug` (required)
2. `name` (required)
3. `profile_hint` (optional; one sentence)

Language support is limited to English and Traditional Chinese.

Output JSON:

```json
{
  "slug": "",
  "name": "",
  "profile_hint": "",
  "language": "en|zh-TW"
}
```
